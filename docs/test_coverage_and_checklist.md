- **覆盖率状态**: 核心模型和 Service 层覆盖率达 90% 以上。

### 1.2 Flutter 组件测试 (Widget Tests)
- **目标**: 验证 UI 组件在不同状态下的渲染是否正确。
- **已实现测试**:

  - `preset_selector_widget_test.dart`: 验证相机选择器的列表渲染和高级相机锁定图标显示。

## 2. 真机联调检查清单 (QA Checklist)

自动化测试无法完全覆盖相机硬件和 GPU 渲染的边缘情况，必须进行严格的真机联调。

### 2.1 核心相机与渲染功能
- [ ] **权限请求**: 首次启动时正确弹出相机、麦克风和相册权限请求。拒绝权限时有合理的 UI 引导。
- [ ] **实时预览**: 画面无明显延迟（目标 60fps），无画面撕裂或绿屏。
- [ ] **滤镜切换**: 切换不同相机（如从 CCD 切换到胶片）时，预览画面实时更新，无闪烁或崩溃。
- [ ] **前后置切换**: 切换前后置摄像头顺畅，画面方向和镜像处理正确。

### 2.2 拍照与录像 (包含相纸逻辑)
- [ ] **拍照**: 点击快门后能正确捕获高分辨率图像。
- [ ] **相纸应用**: **关键逻辑验证**：在拍照前选择相纸后，拍摄的照片在保存到相册时必须**自动带有已选相纸的效果**。拍照后**不应**出现再次选择或应用相纸的二次编辑界面，实现真正的“拍完即得”。
- [ ] **录像**: 长按或切换到录像模式，能正常录制带有滤镜效果的视频并保存。

### 2.3 商业化与订阅 (RevenueCat)
- [ ] **付费墙触发**: 点击带锁的 Premium 相机时，正确弹出订阅页面。
- [ ] **购买流程**: 能够成功调起 App Store / Google Play 的沙盒支付面板并完成购买。
- [ ] **状态刷新**: 购买完成后，UI 立即刷新，Premium 相机解锁可用。
- [ ] **恢复购买**: 点击恢复购买按钮能正确恢复之前的订阅状态。

### 2.4 平台兼容性 (iOS 26 & Android 16)
- [ ] **iOS 26**: 验证新的隐私权限弹窗行为；验证 Metal 渲染管线在最新系统上的稳定性。
- [ ] **Android 16 (API 36)**: 验证 16KB 内存页兼容性（无 Native Crash）；验证 Predictive Back 手势在 Flutter 路由中的表现。

### 2.5 相册与分享
- [ ] **相册加载**: 能正确读取系统相册并展示缩略图。
- [ ] **系统分享**: 选中照片后能正确调起系统分享面板，并成功分享到第三方应用。
